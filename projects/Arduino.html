For arduino, there were two aspects, the hardware aspect was the actual circutry. The other aspect was the code.

For the circuts you have to attatch the ground pin to the ground wire, wich for LEDs is the shorter one, and then the the positive lead to any pin 1-13. But if you would like to make the LED fade in and out, then you hav to use a PMW pin, which is marked by a ~. You also will need a resistor, a 330 ohm resister for LEDs and 10k ohm for buttons, this is so you dont burn out the LED.

For the coding aspect, it is all fairly simple programming. you asigh pins to outputs for LEDs and inputs for buttons. Depending if you use a PMW pin you can either digitally read or use analog read to read the state of the pin. the digital read if either on or off and analog read if for a value of 0 to 255. Analog read is if you want to do a fade with the LED.
